## คืออะไร?

**ConverterIQ** คือโปรแกรม Windows สำหรับแปลงไฟล์วิดีโอ `.qsv` ของแอป iQIYI ให้กลายเป็นไฟล์ `.mp4` ที่เปิดได้ทั่วไป พร้อมดาวน์โหลดซับไตเติ้ลแบบอัตโนมัติในทีเดียว

ไฟล์ `.qsv` คือรูปแบบวิดีโอเฉพาะของ iQIYI ที่ถูก encrypt ไว้ — เปิดด้วยโปรแกรมอื่นไม่ได้โดยตรง ConverterIQ แก้ปัญหานี้

## ความสามารถ

### 🎬 แปลงไฟล์วิดีโอ
- แปลง `.qsv` → `.mp4` โดยใช้ FFmpeg แบบ built-in (ไม่ต้องติดตั้งเพิ่ม)
- รองรับการแปลงแบบ **Batch** ทั้งโฟลเดอร์ในครั้งเดียว
- แสดง progress % ของแต่ละไฟล์แบบ real-time
- รองรับซีรีส์หลายตอน จัดเรียงชื่อให้เป็น `ep01.mp4`, `ep02.mp4`, ...

### 📝 ดาวน์โหลดซับไตเติ้ลอัตโนมัติ
- ดึงซับไตเติ้ลจากเซิร์ฟเวอร์ iQIYI โดยตรง
- รองรับ **12 ภาษา** ได้แก่

  | ภาษา | ภาษา |
  |---|---|
  | 🇨🇳 จีนกลาง (ตัวย่อ) | 🇹🇼 จีนกลาง (ตัวเต็ม) |
  | 🇬🇧 อังกฤษ | 🇰🇷 เกาหลี |
  | 🇯🇵 ญี่ปุ่น | 🇹🇭 ไทย *(ค่าเริ่มต้น)* |
  | 🇲🇾 มาเลย์ | 🇻🇳 เวียดนาม |
  | 🇮🇩 อินโดนีเซีย | 🇪🇸 สเปน |
  | 🇧🇷 โปรตุเกส | 🇸🇦 อาหรับ |

- บันทึกซับไตเติ้ล 3 รูปแบบพร้อมกัน: **`.srt`**, **`.vtt`**, **`.xml`**
- ถ้าภาษาที่เลือกไม่มีในตอนนั้น — โปรแกรมถามให้เลือกภาษาใหม่แล้ว retry อัตโนมัติ

### 🗂️ จัดการไฟล์อัตโนมัติ
- เลือกโฟลเดอร์ output แยกต่างหาก หรือจะให้บันทึกในโฟลเดอร์เดิมก็ได้
- ผลลัพธ์จัดเรียงเป็นโครงสร้างชัดเจน:
  ```
  output/
  ├── mp4/           ← ไฟล์วิดีโอที่แปลงแล้ว
  ├── qsv/           ← ไฟล์ต้นฉบับ (เก็บไว้)
  ├── subtitle_srt/  ← ซับไตเติ้ล .srt
  ├── subtitle_vtt/  ← ซับไตเติ้ล .vtt
  └── subtitle_xml/  ← ซับไตเติ้ล .xml
  ```

### ⬇️ ดาวน์โหลดวิดีโอ iQIYI โดยตรง (Downloader)
- วางลิงก์จาก `iq.com` แล้วดาวน์โหลดได้เลย — ไม่ต้องติดตั้ง yt-dlp แยก
- รองรับ **2 โหมดอัตโนมัติ** ตามประเภท URL ที่ใส่:

  | URL | โหมด |
  |---|---|
  | `iq.com/play/...` | ดาวน์โหลดตอนเดียว (Single EP) |
  | `iq.com/album/...` | ดาวน์โหลดทั้งซีรีส์ทีละตอน (Entire Series) |

- รองรับ **Cookie file** (`.txt`) สำหรับวิดีโอที่ต้องล็อกอิน หรือคอนเทนต์ VIP
- ชื่อไฟล์จัดรูปแบบให้อัตโนมัติ: `EP01 - ชื่อตอน.mp4`, `EP02 - ชื่อตอน.mp4`, ...
- ดาวน์โหลดซับไตเติ้ล `.srt` พร้อมกันในขั้นตอนเดียว
- โหมด Entire Series มีระบบ **retry อัตโนมัติ 3 รอบ** ต่อตอน:
  - รอบ 1: ลองใหม่ทันทีหลัง 30 วินาที
  - รอบ 2: ดึง URL ตรงของตอนนั้น แล้วโหลดแบบ Single
  - ถ้าล้มเหลวทั้งหมด — แจ้งหมายเลขตอนที่พลาดให้โหลดซ้ำเอง
- จำกัดความเร็วที่ **2 MB/s** และพักระหว่างตอน 5–15 วินาที เพื่อลดภาระเซิร์ฟเวอร์
- ผลลัพธ์ถูก move เข้าโฟลเดอร์ย่อยอัตโนมัติหลังโหลดเสร็จแต่ละตอน:
  ```
  output/
  ├── mp4/           ← ไฟล์วิดีโอ .mp4
  └── subtitle_srt/  ← ซับไตเติ้ล .srt
  ```

## วิธีใช้

### แท็บ Converter (แปลงไฟล์ QSV)
1. เปิด `ConverterIQ.exe`
2. กด **Browse...** เลือกโฟลเดอร์ที่มีไฟล์ `.qsv`
3. *(ไม่บังคับ)* เลือกโฟลเดอร์ปลายทางสำหรับผลลัพธ์
4. เลือกภาษาซับไตเติ้ล
5. กด **▶ Run** แล้วรอ

### แท็บ Downloader (ดาวน์โหลดวิดีโอ)
1. เปิด `ConverterIQ.exe` แล้วเลือกแท็บ **⬇ Downloader**
2. วาง URL จาก iQIYI ลงในช่อง **IQIYI VIDEO URL**
   - `iq.com/play/...` → โหลดตอนเดียว
   - `iq.com/album/...` → โหลดทั้งซีรีส์
3. กด **Browse...** เลือกโฟลเดอร์ปลายทาง
4. *(ไม่บังคับ)* เลือก **Cookie File** สำหรับคอนเทนต์ VIP หรือต้องล็อกอิน
5. เลือกภาษาซับไตเติ้ล
6. กด **▶ Start Download** แล้วรอ
